Questions, answered.

Do my payloads go anywhere?

No. Parsing, search, queries, CSV and diff all run inside the viewer tab on your machine. JSONPeek has no servers, no account system, and no analytics.

When can JSONPeek see a page?

Only at the moment you click its icon or press Alt+Shift+J on that tab (Chrome's activeTab permission). It cannot read pages in the background.

How big a file can it handle?

Free parses files up to 2 MB — covering the vast majority of API responses. Pro removes the limit; the lazy tree keeps large payloads responsive by rendering nodes as you expand them.

Is Pro really one-time?

Yes. $12 once, all Pro features forever, every future update included. Payments are processed by Stripe via ExtensionPay; we never see your card details.

Which query syntax is supported?

A practical JSONPath subset: dot keys ($.a.b), bracket keys ($["odd key"]), indexes including negative ([0], [-1]), wildcards ([*], .*), and recursive descent ($..email). That covers day-to-day API spelunking.
