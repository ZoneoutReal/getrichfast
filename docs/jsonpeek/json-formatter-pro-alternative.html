When another tool is the better choice: if you want every JSON URL to auto-format the instant it loads, use one of the free always-on formatters — that requires reading all sites, which JSONPeek deliberately avoids (it's invoke-only via activeTab). And if you only ever pretty-print small responses, a free formatter is all you need.

What JSONPeek does differently

  • Invoke-only access. One click loads the tab's JSON. No always-on content script reading every page you visit.
  • Answers, not scrolling. $..email finds every email in a payload; $.items[*].price pulls a column. Then export it as CSV.
  • "What changed?" built in. Paste staging and prod payloads, get the changed paths color-coded.
  • One price. $12 once. The subscription option costs that every 10 weeks.
